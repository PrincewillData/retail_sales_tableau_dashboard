Retail Sales Dashboard (Tableau)

Objective  
Analyze sales performance to identify top-performing products and monthly trends.

Tools Used  
Tableau, Excel

Project Structure  
- data: dataset used for analysis  
- dashboard: Tableau dashboard file  
- images: dashboard preview  

Process  
- Cleaned dataset in Excel  
- Created calculated field: Sales = Quantity × Unit Price  
- Built bar chart for product performance  
- Built line chart for monthly sales trend  
- Added category filter for interactivity  
- Designed dashboard for clear insights  

Key Insights  
- Laptops generated the highest revenue  
- Sales peaked in November  
- Low-performing products contribute minimally to total revenue
